

You can invoke this command writing 'help' in the commands from any directory.

- 'makeFile': ()

NOTE: Prompt.py isn't case-sensitive for commands' names, camelCase is used in order to increment readability

HELP:

help [command => (default='help')]

command <compulsory-argument: type> [optional-argument: type => (default-value)] <{argument-taken-by-input: str}>
output
information about the command

NOTE: If you want help for a command that has aliases, you MUST use the original name of the command
es. 'help setVar' MUST be used instead of 'help @'
